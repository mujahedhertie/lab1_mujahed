Measurements errors

  1. Systematic Error (Validity)
    • Face validity
    • Construct validiy
  2. Random Error (Reliability)

Levels of measurements

  1. Nominal (Simply categorical data with no ranking or ordering )

  2. Ordinal (Simply categorical data with ranking or ordering )

  3. Interval (A measurement where the difference between two values is significant)
